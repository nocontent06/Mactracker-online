## 📖 About
 
Mactracker Online brings the spirit of Ian Page's beloved **Mactracker** app to the web. It's a fast, searchable database of Apple hardware — Macs, and other Apple-branded devices — built with a modern UI and accessible directly in your browser.
 
No installer, no platform restrictions: just open a tab and search.
 
## ✨ Why Mactracker Online?
 
| | Mactracker (Original) | Mactracker Online |
|---|---|---|
| 🌐 **Platform support** | macOS / iOS only | Any OS with a browser |
| 🎨 **Interface** | Native app UI | Modern, redesigned UI |
| 🔓 **Source** | Closed source | Fully open source |
| 💻 **Installation** | Required | Not required (web) — optional desktop builds available |
 
If you'd still prefer a native experience, standalone executables for **macOS** and **Windows** are available under [Releases](../../releases).
